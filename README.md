# Klasifikasi Lahan Sawah Berbasis NDVI Menggunakan LSTM

Aplikasi ini merupakan implementasi model **Long Short-Term Memory (LSTM)**
untuk klasifikasi **lahan sawah** dan **non-sawah** berbasis **deret waktu NDVI (2005â€“2024)**.

## Fitur
- Upload data NDVI dalam format CSV
- Visualisasi grafik NDVI multitemporal
- Prediksi lahan sawah / non-sawah
- Interpretasi hasil klasifikasi

## Model
- Arsitektur: LSTM
- Input: NDVI tahunan
- Output: Sawah / Non-Sawah

## Deployment
Aplikasi ini dideploy menggunakan **Streamlit Cloud**.
